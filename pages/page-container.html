Page-container

Page-container is the primary container for FINN pages. It is centered and has a fixed-width. It is considered the outer boundary for content.

Warning

Note that page-container is not fully compatible with panel from Troika. Please migrate to the new Box component in Fabric before using this class, or ensure your Panels still have the appropriate vertical-alignment.

Common usage

 Note that the page-container class doesn't have to be applied to a main element. You can adjust this as needed.
